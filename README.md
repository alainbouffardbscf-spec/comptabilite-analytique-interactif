# Comptabilité analytique interactif

Application web pédagogique en **HTML, CSS et JavaScript pur** destinée à l’enseignement de la **comptabilité analytique de gestion**.

Le site permet de sélectionner des écritures comptables et d’en visualiser l’impact en temps réel sur plusieurs états comptables et analytiques.

## Fonctionnalités

L’application permet de :

- charger un plan comptable en JSON ;
- charger un exercice en JSON ;
- cocher ou décocher des écritures comptables ;
- afficher les soldes d’ouverture ;
- afficher les comptes en T ;
- construire une balance de vérification ;
- produire un bilan ;
- produire un état des résultats ;
- produire un état du coût de fabrication.

## Technologies

- HTML
- CSS
- JavaScript
- JSON

Aucun framework n’est utilisé.
